DEPLOYMENT INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Get Telegram Bot Token
  1. Message @BotFather on Telegram
  2. Send: /newbot
  3. Follow prompts, name your bot
  4. Copy bot token (example: 123456789:ABCdefGHIjklmnoPQRstuvWXYZabcdefGH)

STEP 2: Get Telegram Chat ID
  1. Create Telegram group or use existing
  2. Add bot to group
  3. Visit: https://api.telegram.org/botTOKEN/getUpdates
  4. Replace TOKEN with your bot token
  5. Find "chat" → "id" (example: -1001234567890)

STEP 3: Set Environment Variables in Vercel
  1. Go to Vercel project dashboard
  2. Settings → Environment Variables
  3. Add: TG_BOT_TOKEN = (your token from Step 1)
  4. Add: TG_CHAT_ID = (your chat ID from Step 2)
  5. Save and wait 60 seconds

STEP 4: Deploy
  1. Push code to repository (or redeploy from Vercel)
  2. Vercel auto-deploys with env variables
  3. Check deployment complete (green checkmark)
  4. Visit your app URL

STEP 5: Test
  1. Open application
  2. Fill form (test data is fine)
  3. Add dummy files (selfie, ID photos)
  4. Submit
  5. Check Telegram group for message + files
