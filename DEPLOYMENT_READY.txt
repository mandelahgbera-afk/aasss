TROUBLESHOOTING GUIDE
═════════════════════════════════════════════════════════════════════════════

Issue: Form won't submit
Solution: Check TG_BOT_TOKEN and TG_CHAT_ID are set
          Check browser console for errors (F12)
          Verify internet connection

Issue: Files not uploading
Solution: Check file size < 4MB
          Check total payload < 3.5MB
          Verify file format is supported
          Clear browser cache and retry

Issue: Validation not working
Solution: Ensure JavaScript is enabled
          Check browser console for errors
          Verify form HTML is loaded

Issue: Offline mode not showing
Solution: Use DevTools > Network > Offline
          Check window.offline event listener
          Verify browser supports online/offline events

Issue: Telegram not receiving message
Solution: Verify TG_BOT_TOKEN is correct (check bot token)
          Verify TG_CHAT_ID is correct (check chat ID)
          Test bot manually with curl command
          Check Telegram API response in console
